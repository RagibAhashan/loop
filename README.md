# sneaker

Install dependencies :

`npm install`

Run prototype :

`node main.js` _or_ `npm start`

## To set credit card info as environment variables in windows powershell


POWERSHELL:
```
$env:NUMBER="4242 4242 4242 4242"
$env:YEAR="2030"
$env:MONTH="04"
$env:CVC="123"
```

MAC/LINUX:
```
export NUMBER="4242 4242 4242 4242"
export YEAR="2030"
export MONTH="04"
export CVC="123"
```
